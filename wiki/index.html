Building for Linux

An unofficial PKGBUILD of Citra is available for Arch Linux on the AUR. Dependencies: You’ll need to download and install the following to build Citra: SDL2 Deb: apt install libsdl2-dev Arch: pacman -S sdl2 Fedora: dnf install SDL2-devel OpenSUSE: zypper in libSDL2-devel OpenSSL (optional) Deb: apt install libssl-dev Arch: pacman -S openssl-1.0 Fedora: dnf install openssl-devel OpenSUSE: zypper in openssl-devel Qt 6.2+ Only 6.2+ versions are tested. Lower version might or might not work.

Building for Windows
